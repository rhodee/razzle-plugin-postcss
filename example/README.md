# Razzle x With PostCSS

## How to use

Download the example [or clone the whole project](https://github.com/rhodee/razzle-plugin-postcss.git):

```bash
curl https://codeload.github.com/rhodee/razzle-plugin-postcss/tar.gz/master | tar -xz --strip=2 razzle-plugin-postcss-master/example
cd example
```

Install it and run:

```bash
yarn install
yarn start
```

## Idea behind the example

This is a basic, bare-bones example of Razzle with PostCSS updated to support SCSS and any other PostCSS plugins you know how to configure.
